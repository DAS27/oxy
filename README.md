## Description
API for Blog

## Requirements
- PHP ^7.3
- **[Laravel sail](https://laravel.com/docs/8.x/sail)** for local
- **[Composer](https://getcomposer.org/)**
- **[Postman](https://www.postman.com/)**

## Setup
````
$ make setup
````

## Launch local
Server will be available at http://localhost
````
$ make start
````
